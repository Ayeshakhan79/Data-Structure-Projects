**Data Structures Projects:**
This repository contains two Java console applications demonstrating basic data structures:

## 1. Car Sales System
- **Purpose:** Manage car inventory using a linked list.
- **Features:**
  - Add, update, delete, and sell cars.
  - Display and sort inventory by brand.
  - Handles new and used cars with registration and mileage details.

## 2. Student Information System
- **Purpose:** Manage student records using a linked list.
- **Features:**
  - Add student details and marks.
  - Update student marks.
  - Search student by name or roll number.
  - Print marksheets and full student list.
  - Sort students alphabetically by first name.
