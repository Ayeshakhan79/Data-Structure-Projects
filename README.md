**Data Structures Projects**:
This repository contains two Java console applications demonstrating data structures:

## 1. Car Sales System
- Manage car inventory using linked lists.
- Features: Add, update, delete, sell, display, and sort cars.
- Handles both new and used cars with registration and kms details.

## 2. Student Information System
- Manage student records using linked lists.
- Features: Add student, update marks, search student, print marksheet, print student list, sort students.
- Tracks student details and marks for multiple subjects.
